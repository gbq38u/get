# Raspberry Pi: ADC / DAC / GPIO Practice

Учебный проект по работе с **Raspberry Pi**, **GPIO**, **светодиодами**, **ADC/ЦАП**, **DAC/АЦП** и базовыми интерфейсами взаимодействия с внешними компонентами.

Проект выполнен в рамках университетской практики и объединяет набор лабораторных и прикладных задач на Python: работа с цифровыми выводами, светодиодами, измерением аналоговых сигналов, генерацией напряжения и базовой обработкой результатов.

## Что реализовано

В репозитории собраны скрипты и задания по следующим темам:

- управление GPIO на Raspberry Pi
- работа со светодиодами и индикаторами
- базовые схемы с кнопками и цифровыми входами/выходами
- измерение аналогового сигнала через АЦП
- генерация уровней напряжения через ЦАП
- пошаговое изменение напряжения и анализ отклика схемы
- простые эксперименты по дискретизации и преобразованию сигналов
- базовая автоматизация экспериментов на Python

## Цели проекта

- освоить практическую работу с GPIO на Raspberry Pi
- понять принципы работы **аналого-цифрового преобразования**
- понять принципы работы **цифро-аналогового преобразования**
- научиться управлять внешними электронными компонентами с Python
- закрепить базовые навыки работы с низкоуровневым взаимодействием между кодом и аппаратурой

## Используемый стек

- **Python**
- **Raspberry Pi**
- **RPi.GPIO** / аналоги библиотек для GPIO
- макетная плата, светодиоды, резисторы
- схемы с АЦП/ЦАП
- базовые электронные компоненты
